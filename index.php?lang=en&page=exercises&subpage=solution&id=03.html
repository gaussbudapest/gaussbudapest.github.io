Solution (National Theatre)

First, we have to make a good sketch of the problem, which later helps us:

We can see two triangles in the picture. Both of them are right triangles. What does the right triangle remind us of?

A. Thales-theorem B. Pythagorean-theorem C. trigonometric functions

In this case we know angles, so we have to use the trigonometric functions. We know something about the adjacent legs and we are looking for the opposite legs. So we have to use either the tangent or the cotangent functions. Let's symbolize the height of the theatre by h, and the first distance to the theatre by x, both measured in meter.
