Megoldás (Országos Széchényi Könyvtár)

Ez egy olyan számtani sorozat, amelynek első eleme a1 = 23, a differencia pedig d = 5.

A polcok száma n = 8.

A számtani sorozat összegképletére vonatkozó összefüggést használva:

Sn =
n
2
 · (2a1 + (n − 1) · d)

Behelyettesítve:

Sn =
8
2
 · (2 · 23 + 7 · 5) = 324

Összesen 324 db könyv van a polcokon.
